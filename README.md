# CineFind — Web Movie & TV Show Finder

**CineFind** is a modern and responsive web application designed to help users search for movies, TV shows, and cartoons while accessing comprehensive details about them. The project focuses on speed, an intuitive user interface, and an optimal user experience.

---

## 🚀 Features

* **Smart Search:** Quickly find any movie or TV show by its title in real time.
* **Detailed Insights:** Displays complete information, including plot summaries, ratings, cast, release year, and trailers.
* **Recommendation Engine:** Automatically suggests similar titles, making it easy to decide what to watch next.
* **Dynamic Theme Switcher (Dark/Light Mode):** Comfortable viewing at any time of day—toggle between dark and light themes with a single click.
* **Persistent Preferences:** The selected theme is saved in `localStorage`, ensuring your preference is remembered even after reloading the page or reopening the browser.

---

## 🛠️ Tech Stack

This project is built using a clean, vanilla web stack without the overhead of heavy frameworks:

* **HTML5** — For semantic web page structure.
* **CSS3** — For modern styling, responsive layouts across all devices, and smooth transition animations.
* **JavaScript (ES6+)** — Powers the entire core logic: handling search queries, dynamically updating the DOM, managing recommendation features, and controlling the UI theme state with `localStorage` integration.

---

## 🌓 How the Theme Toggle Works

The theme switching logic is handled entirely through JavaScript. When the toggle button is clicked, the application dynamically modifies classes or CSS variables to update the color palette instantly. To enhance the user experience, the application utilizes the **Web Storage API (`localStorage`)** to save the user's choice, preventing the theme from resetting on page refresh.
